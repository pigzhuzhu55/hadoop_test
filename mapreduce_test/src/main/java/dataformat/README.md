## 07-自定义数据格式

- 输入每科学员的成绩单
- 输出学员的总成绩、平均成绩

- 比如：
>  data.txt
```html
19020090040 秦心芯 123 131 100 95 100
19020090006 李磊 99 92 100 90 100
```

- 结果：
```html
19020090040	秦心芯	549.0	109.8
19020090006	李磊	481.0	96.2
```
